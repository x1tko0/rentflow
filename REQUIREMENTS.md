# RentFlow — Техническое задание

## 1. Общее описание

**Продукт:** Мобильное приложение для управления арендой недвижимости  
**Целевая аудитория:** Частные арендодатели и небольшие портфели недвижимости: 1 объект в Free, до 20 объектов в Pro  
**Платформы:** iOS, Android (Flutter)  
**Язык интерфейса:** Русский, Английский

## 2. Проблема

Существующие решения (Rentec Direct, Buildium, AppFolio) стоят $30–150/мес и ориентированы на управляющие компании. Для частного арендодателя с несколькими объектами или небольшим портфелем до 20 объектов это часто избыточно и дорого. Excel-таблицы неудобны и не дают аналитики.

## 3. Решение

Простое, красивое приложение с фокусом на:
- Быстрый ввод платежей (2 тапа)
- Наглядная аналитика доходов/расходов
- Готовые отчёты для налоговой
- Работа offline

## 4. Функциональные требования

### 4.1 Управление объектами
- [ ] Добавление/редактирование/удаление объекта
- [ ] Фото объекта
- [ ] Базовая информация: адрес, площадь, тип
- [ ] История владения (цена покупки, дата)

### 4.2 Управление арендаторами
- [ ] Карточка арендатора: ФИО, контакты
- [ ] Условия аренды: сумма, срок, депозит
- [ ] Привязка к объекту
- [ ] История арендаторов по объекту
- [ ] Статус: активный / бывший

### 4.3 Учёт платежей
- [ ] Календарь ожидаемых платежей
- [ ] Быстрая отметка "Оплачено"
- [ ] Частичная оплата
- [ ] Просроченные платежи (выделение цветом)
- [ ] История платежей по арендатору/объекту
- [ ] Типы: аренда, депозит, коммуналка, прочее

### 4.4 Учёт расходов
- [ ] Категории: ремонт, налоги, страховка, ЖКХ, управление
- [ ] Привязка к объекту
- [ ] Фото чека/квитанции
- [ ] Повторяющиеся расходы

### 4.5 Аналитика и отчёты
- [ ] Dashboard: доход/расход за месяц, год
- [ ] График денежного потока
- [ ] Доходность по объектам
- [ ] Годовой отчёт для налоговой (PDF)
- [ ] Экспорт данных (CSV)

### 4.6 Напоминания
- [ ] Push о предстоящем платеже (за 3 дня)
- [ ] Push о просроченном платеже
- [ ] Напоминание об окончании договора

### 4.7 Настройки
- [ ] Выбор валюты
- [ ] Biometric lock
- [ ] Резервное копирование
- [ ] Тёмная тема

## 5. Нефункциональные требования

### 5.1 Производительность
- Запуск приложения < 2 сек
- Плавные анимации 60 fps
- Размер APK < 30 MB

### 5.2 Offline-first
- Все функции работают без интернета
- Синхронизация при появлении сети (Pro)

### 5.3 Безопасность
- Шифрование локальной БД
- Опциональная биометрия
- Нет передачи данных третьим лицам

### 5.4 UX
- Онбординг для новых пользователей
- Пустые состояния с подсказками
- Минимум шагов для частых действий
